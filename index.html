Building HTML Lists

Embedded Style Sheet
- A method of including CSS directly within an HTML document, inside the "style" tags located in the "head" section.
External Style Sheet
- A separate file containing CSS rules that define the presentation and formatting of one or more HTML documents.
Inline Style
- The application of CSS styling directly to an individual HTML element using the style attribute within the element's opening tag.
